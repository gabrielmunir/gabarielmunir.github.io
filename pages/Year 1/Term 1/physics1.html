1.5 - Estimates and Order-of-Magnitude Calculations

If you need to approximate a value to an order of magnitude:

  1. Express the value in scientific notation.

    2. $a\times 10^{n}$

  2. Use $\sqrt{10} = 3.162$ as a baseline.

    3. If $a>3.162$, then $n=$ Order of Magnitude.

    If $a\leq3.162$, then $n+1=$ Order of Magnitude.
